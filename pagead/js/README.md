# AdsByGoogle.js Integration Script

This repository contains a JavaScript snippet for dynamically loading and configuring Google AdSense ads on your website. The script allows you to set ad client IDs, host IDs, page URLs, and ad dimensions via URL parameters, with fallback default values.

## Demo

Check out the live demo here: [https://joe-x5.github.io/pagead/js/adsbygoogle.js](https://joe-x5.github.io/pagead/js/adsbygoogle.js)

## Features

- Dynamic configuration through URL parameters
- Default fallback values
- Easy integration into your webpage

## Usage

Include the script in your webpage and pass parameters via URL:

```html
<script src="https://joe-x5.github.io/pagead/js/adsbygoogle.js"></script>
